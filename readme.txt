BSPlaces extends WordPress of simple gps position and gpx tracks management. Default visualization engine is api.mapy.cz.

== Description ==

BSPlaces plugin adds possibility to specify gps position or attach gpx track files for post or page. All gps data are
shown on map which is generated automatically for all posts and pages with attached any gps information. 

== Installation ==

1. Install and activate the plugin through the 'Plugins' menu in WordPress.
2. That's it! :)
